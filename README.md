# Fauna Morocco — Éditeur en ligne (paquet Render)

Remplace le contenu de ton dépôt **fauna-editor-2** par ces fichiers, puis :
`commit` → `push` → Render redéploie automatiquement (~2 min).
Aucun nouveau réglage : `EDITOR_PASSWORD`, `PUBLISH_PASSWORD`, `GITHUB_TOKEN` sont déjà sur Render.

## Adresses (une fois en ligne)
- Tableau de bord : `https://fauna-editor-2.onrender.com/app/dashboard.html`
- Éditeur : `https://fauna-editor-2.onrender.com/app/wysiwyg.html`

## Ce qui est branché
- Éditeur → **Enregistrer** (brouillon) et **Publier** (écrit la fiche EN/FR/AR dans le dépôt du site → déploiement Hostinger auto).
- **Upload d'images** : glisser-déposer → compression navigateur → envoi au dépôt.
- Tableau de bord → **Enregistrer sur le serveur** (liste des espèces) + chargement auto au démarrage.
- Une langue n'est publiée que si elle a un nom (pas de page arabe vide).
